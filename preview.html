Django SinglePage 应用预览

欢迎来到 SinglePage 应用

这是一个单页面Django应用的预览版本。

完整功能包括：

  • 动态内容加载
  • URL路由管理
  • 模板系统
  • Django后台管理

安装说明：

  1. 下载并安装 Python：https://www.python.org/downloads/
  2. 安装依赖：pip install -r requirements.txt
  3. 启动服务器：python manage.py runserver 8000
  4. 访问：http://localhost:8000/
